# Data Structures & Algorithms (DSA)

This repository contains my practice and learning of **Data Structures and Algorithms** using C++.

---

## 🎯 Objective

* Build strong problem-solving skills
* Understand core data structures and algorithms
* Learn to write efficient and optimized code
* Prepare for coding interviews

---

## 📚 Topics Covered

* Arrays & Strings
* Linked List
* Stack & Queue
* Trees
* Graphs
* Recursion & Backtracking
* Dynamic Programming
* Searching & Sorting

---

## 🔁 Problem Solving Approach

For each problem:

1. Understand the problem clearly
2. Start with a simple (brute force) solution
3. Optimize step by step
4. Focus on time and space complexity
